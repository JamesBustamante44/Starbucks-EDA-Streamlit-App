
# Starbucks EDA Streamlit App

Interactive EDA app for Starbucks offer events and customer demographics. Upload CSVs or use the included synthetic sample to explore funnels, timelines, and demographics.

## Quick start (local)

```bash
pip install -r requirements.txt
streamlit run app.py
```

Place your data next to app.py (e.g., `cleaned_starbucks.csv`) or upload via the UI.

## Deploy on Streamlit Cloud
1. Push this repo to GitHub.
2. In Streamlit Cloud, Deploy app and set main file to `app.py`.

## Data
- `data/cleaned_starbucks.csv`: a tiny synthetic dataset for demo.
